# 小帅 AI 日报 - 功能介绍文档

---

## 一、产品概述

**小帅 AI 日报** 是一款微信小程序，专注于聚合全球 AI 领域最新资讯，通过 AI 智能处理为用户提供高质量的新闻阅读体验。

### 1.1 核心定位
- **定位**：AI 新闻聚合平台
- **目标用户**：AI 爱好者、从业者、学生
- **价值主张**：一站式获取 AI 最新资讯，AI 智能摘要，节省阅读时间

### 1.2 技术亮点
- 多源信息聚合（17+ 信息源）
- AI 智能摘要生成
- 社区评论智能总结
- 三种阅读主题模式
- 个性化收藏功能

---

## 二、核心功能模块

### 2.1 首页 - 新闻列表

#### 2.1.1 功能描述
首页是用户的主要浏览页面，展示按日期聚合的 AI 新闻。

#### 2.1.2 具体功能

| 功能 | 说明 |
|------|------|
| **日期切换** | 支持查看过去 7 天的新闻 |
| **下拉刷新** | 下拉页面刷新当前日期的新闻 |
| **新闻列表** | 瀑布流展示，包含标题、来源、发布时间 |
| **新闻数量统计** | 顶部显示当日新闻总数 |
| **主题切换** | 右上角按钮切换主题模式 |

#### 2.1.3 界面元素
- 顶部导航：Logo、日期选择器、主题切换按钮
- 新闻卡片：标题、来源 Logo、发布时间、摘要预览
- 底部 TabBar：首页、收藏、我的

---

### 2.2 详情页 - 新闻阅读

#### 2.2.1 功能描述
展示单条新闻的详细内容，包含 AI 摘要、原文、社区评论等。

#### 2.2.2 具体功能

| 功能 | 说明 |
|------|------|
| **新闻标题** | 大字体展示新闻标题 |
| **来源信息** | 显示信息源名称、发布时间、原文链接 |
| **AI 摘要** | 由火山引擎 AI 生成的智能摘要 |
| **原文内容** | 新闻完整原文（非中文时保持原文） |
| **社区评论** | Hacker News/Reddit 等社区的热门评论总结 |
| **主题切换** | 顶部导航切换阅读模式 |

#### 2.2.3 操作按钮

| 按钮 | 功能 |
|------|------|
| **分享给好友** | 一键分享新闻到微信好友 |
| **复制链接** | 弹出对话框显示原文链接，支持一键复制 |
| **收藏** | 收藏/取消收藏该新闻 |

---

### 2.3 收藏页 - 个人收藏

#### 2.3.1 功能描述
展示用户收藏的所有新闻，支持管理收藏。

#### 2.3.2 具体功能

| 功能 | 说明 |
|------|------|
| **收藏列表** | 展示用户收藏的所有新闻 |
| **取消收藏** | 点击取消收藏按钮移除收藏 |
| **查看详情** | 点击卡片进入详情页 |
| **空状态提示** | 无收藏时显示提示文字 |

---

### 2.4 我的 - 个人中心

#### 2.4.1 功能描述
用户个人信息展示和基础设置。

#### 2.4.2 具体功能

| 功能 | 说明 |
|------|------|
| **用户信息** | 显示微信头像、昵称 |
| **统计信息** | 展示收藏数、阅读天数等 |
| **主题切换** | 快速切换三种阅读模式 |
| **关于我们** | 项目介绍和联系方式 |

## 三、主题模式系统

### 3.1 三种主题模式

#### 3.1.1 白天模式（默认）
- 背景：浅灰白渐变
- 卡片：白色
- 文字：深色
- 适合场景：白天阅读、明亮环境

#### 3.1.2 夜晚模式
- 背景：深蓝黑渐变
- 卡片：深蓝灰
- 文字：浅灰色
- 适合场景：夜晚阅读、护眼需求

#### 3.1.3 护眼模式
- 背景：暖黄渐变
- 卡片：米白色
- 文字：棕褐色
- 适合场景：长时间阅读、护眼需求

### 3.2 主题切换特性

| 特性 | 说明 |
|------|------|
| **全局同步** | 在任一页面切换主题，所有页面自动同步 |
| **本地持久化** | 主题设置保存在本地，下次打开自动生效 |
| **平滑动画** | 主题切换时带有平滑过渡动画 |
| **快捷切换** | 首页和详情页都有主题切换按钮 |

## 四、AI 智能处理

### 4.1 新闻摘要生成

#### 技术实现
- 使用火山引擎大模型 API
- 基于新闻内容智能生成摘要
- 非中文内容生成中文摘要，原文保持原样

#### 摘要特点
- 简洁：控制在 200-300 字
- 准确：保留核心信息
- 易读：语言流畅自然

### 4.2 社区评论总结

#### 功能说明
- 抓取 Hacker News、Reddit 等社区的热门评论
- AI 智能总结评论观点
- 提取讨论焦点和争议点

### 4.3 信息筛选

#### 关键词过滤
- 预设 AI 相关关键词库
- 自动过滤非 AI 相关内容
- 确保资讯质量

## 六、技术特色功能

### 6.1 数据缓存机制

- Redis 缓存首页新闻列表
- 缓存过期时间：1 小时
- 爬虫更新时自动清除缓存
- 缓存命中响应时间：< 10ms

### 6.2 定时任务

- 每天凌晨 2 点自动执行爬虫
- 支持手动触发爬虫
- 时区自动同步（北京时间）

### 6.3 用户系统

- 微信授权登录
- 收藏数据与用户关联
- 个人设置本地存储
